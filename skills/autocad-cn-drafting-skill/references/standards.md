# Standards Baseline

This Skill contains original implementation guidance, not standard text. Obtain
official editions and confirm local adoption before setting a finding to `code`.

## Drafting References

- `GB/T 50001-2017`, Unified standard for building drawings.
- `GB/T 50104-2010`, Standard for architectural drawings.
- `GB/T 14689`, Technical drawings - Size and layout of drawing sheets.
- `GB/T 14691`, Technical drawings - Lettering.
- `GB/T 14665`, Mechanical engineering drawings - Basic conventions for CAD
  drawings. Use only where relevant to the organization's CAD management rules.

## Space and Accessibility References

- `GB 55019-2021`, General code for accessibility of buildings and municipal
  engineering.
- `GB 55031-2022`, General code for civil buildings.
- `GB 50096-2011`, Design code for residential buildings, subject to current
  project applicability and later amendments/replacements.

Consult official sources:

- Ministry of Housing and Urban-Rural Development:
  <https://www.mohurd.gov.cn/>
- National Standard Information Public Service Platform:
  <https://std.samr.gov.cn/>
- National Open Standards platform:
  <https://openstd.samr.gov.cn/>

## Governance

Before adopting or changing a numeric rule:

1. Identify the standard edition and clause.
2. Confirm whether it is mandatory, recommended, local, or contractual.
3. Confirm the building type and project stage.
4. Store the source and applicability decision with the project.
5. Get review from the responsible architect or interior designer.

Default ergonomic values in `assets/default_rules.json` are design baselines
unless explicitly documented otherwise. They must not be presented as automatic
legal compliance.
